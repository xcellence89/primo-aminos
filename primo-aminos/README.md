# Primo Aminos (RUO) storefront

Minimal, compliance-forward Next.js site for a research-use-only (RUO) peptide catalog.

## Quick start
```bash
npm install
npm run dev
```

## Edit products & Janoshik links
Update `src/lib/catalog.ts`.

## Deploy (recommended)
1. Create a private GitHub repo
2. Upload this project (all files)
3. Import the repo into Vercel
4. Add domains: primoaminos.com and www.primoaminos.com
5. Copy the DNS records Vercel provides into GoDaddy
